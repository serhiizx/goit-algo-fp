## Завдання 1: Однозв'язний список

**Опис:** Реалізація операцій над однозв'язним списком.

**Реалізовано:**
- Реверс списку
- Сортування злиттям (merge sort)
- Об'єднання двох відсортованих списків

**Результат:**
```
Зв'язний список: 45 -> 12 -> 78 -> 34 -> 56 -> 23 -> 89 -> 67
Після реверсу:   67 -> 89 -> 23 -> 56 -> 34 -> 78 -> 12 -> 45
Після сортування: 12 -> 23 -> 34 -> 45 -> 56 -> 67 -> 78 -> 89
```

## Завдання 3: Алгоритм Дейкстри

**Опис:** Пошук найкоротших шляхів у зваженому графі з використанням бінарної купи.

**Складність:** O((V + E) log V)

**Результат:**
```
Найкоротші шляхи від вершини 'A':
A -> B: відстань = 5,  шлях: A -> B
A -> C: відстань = 10, шлях: A -> C
A -> D: відстань = 8,  шлях: A -> B -> D
A -> E: відстань = 12, шлях: A -> B -> D -> E
```

## Завдання 4: Бінарна купа (Max-Heap)

**Опис:** Перетворення довільного бінарного дерева на max-heap.

**Реалізовано:**
- Конвертація дерева в масив (BFS)
- Побудова max-heap з масиву
- Присвоєння значень назад у дерево
- Кольорова візуалізація по рівнях

## Завдання 7: Метод Монте-Карло

**Опис:** Симуляція 1,000,000 кидків двох кубиків для визначення ймовірностей сум.

**Результат:**

| Сума | Монте-Карло | Аналітична | Різниця |
|:----:|:-----------:|:----------:|:-------:|
| 2 | 2.78% | 2.78% | ~0% |
| 7 | 16.63% | 16.67% | -0.04% |
| 12 | 2.77% | 2.78% | -0.01% |

![Результат симуляції](task7/result.png)

**Висновок:** При 1 млн ітерацій похибка < 0.1%. Метод Монте-Карло ефективний для оцінки ймовірностей.
